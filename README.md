# FantasySilk
### FantasySilk is a user guided hub for everything fantasy football. It allows for the creation of players/teams and also provides information and statistics on said platforms.
---
## Technologies Used
- HTML/EJS
- CSS
- javaScript
- MongoDB
- Mongoose
- Google Fonts

- Heroku
- Express
- Node.js
